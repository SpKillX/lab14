# Prompt Log

## Задание М1, M2, M3: Разработка сборщика данных на Go. Добавление буферизации и пакетной записи. Реализация graceful shutdown.

### Промпт 1

**Инструмент:** Gemini

**Промпт:** "Напиши Go-сборщик для эмуляции получения данных API ГИБДД (ДТП). Нужны горутины, канал с буфером, пакетная запись в JSON Lines (по 50 записей) и graceful shutdown через перехват SIGINT/SIGTERM."
**Результат:** Получен рабочий код с использованием `signal.NotifyContext` и `sync.WaitGroup`. Данные корректно пишутся батчами и файл дозаписывается при закрытии канала.

### Итого

- Количество промптов: 1

- Что пришлось исправлять вручную: Ничего.

- Время: 10 мин
